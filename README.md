# normal-db
Repository for project work for the course CS315: Principles of Database Systems.

# Setting up
You will need a recent version of Python. We have tested
with the latest version of Python, *i.e.*, Python 3.14.4.

In the root directory of the cloned repository, create a
virtual environment and run
```bash
pip install .[gui]
```
to install the package dependencies.

To launch the app on `localhost`, run
```bash
flask run
```
on the command line.
